
A simple CLI based game, based on the popular 15-tile puzzle.

Usage:
    tile_puzzle
    tile_puzzle [OPTIONS]

Help Options:
    -s , --size [INT]   Starts game with [INT] sized square game board.
    -r, --rows [INT]    Starts game with [INT] number of rows. Overwrites --size.
    -c, --cols [INT]    Starts game with [INT] number of cols. Overwrites --size.
    -d, --debug         Enable debugging mode.
    -h, --help          Prints this help menu.

Controls:
    WASD    Movement controls.
    q       Quit game.
    `       Toggle debugging mode
